# Рабочий Telegram-бот «Фабрика Деталей» — версия 3.1.0

Бот предназначен для небольшой мастерской: объекты клиентов, задачи сотрудников, внутренние дела, финансы и рабочее портфолио находятся в одной SQLite-базе.

## Что добавлено в версии 3

### 📦 Объекты

Объект создаётся быстро: сначала бот запрашивает только категорию и название. После этого объект можно создать сразу или открыть «Добавить подробности» и заполнить адрес, описание, контакты заказчика, договорённость, ответственного, срок, шаблон задач и начальный статус.

В разделе объектов доступны последние 5 открытых объектов, компактный список активных и отдельный список завершённых.

В карточке объекта доступны задачи, статус, контакты заказчика, договорённости, комментарии, фотографии и финансы.

Статусы объекта:

```text
🆕 Новый
📐 Замер
📏 Проектирование
🔩 Изготовление
🎨 Покраска
🛠 Монтаж
✅ Завершён
⏸ Приостановлен
❌ Отменён
```

### 📋 Шаблоны задач

Для заборов, навесов и ворот предусмотрены готовые наборы задач. Задачи создаются после подтверждения объекта и получают ответственного и автоматические напоминания.

### ✅ Мои задачи

Сотрудник видит только назначенные ему задачи:

```text
🔥 На сегодня
📅 Ближайшие
⚠️ Просроченные
▶️ В работе
✅ Выполненные
```

К задаче можно добавить комментарий и фотографию. Фото задачи, связанной с объектом, также попадает в фотографии объекта и портфолио.

### 📝 Дела мастерской

Внутренние работы не смешиваются с клиентскими объектами.

Категории:

```text
🔧 Оборудование
🏭 Цех и помещение
🚗 Транспорт
💳 Платежи
🧹 Хозяйственные дела
📌 Прочее
```

Поддерживаются ответственный, срок, приоритет, комментарии, фотографии, расходы и повторение раз в неделю или месяц.

### 💰 Финансы

К расходу можно прикрепить одно или несколько фото чека. Фотографии открываются из карточки расхода, а количество фото попадает в CSV-выгрузку. Описание расхода необязательно.

Расход можно привязать:

- к объекту;
- к делу мастерской;
- оставить без привязки.

Категории расходов:

```text
🧱 Материалы
🛠 Инструмент
🏢 Аренда
🧹 Хозтовары
📣 Реклама
📌 Прочие расходы
```

Все сотрудники могут добавлять расходы. Доходы доступны только администраторам и привязываются к объектам.

Администратору доступна кнопка `📤 Выгрузить финансы`. Бот создаёт CSV-файл, который открывается в Excel. В файл входят доходы, расходы, итоговая разница и подробный список операций за выбранный период.

### 📸 Портфолио

Все фотографии объекта автоматически доступны:

- по объектам;
- общей лентой по категориям;
- в недавно добавленных.

Администратор вручную отмечает фотографии как:

```text
⭐ Лучшее
🌐 Для сайта
⬅️ До
➡️ После
```

Файл не дублируется: сортировка выполняется по данным в базе.

### 👥 Сотрудники

Исправлено управление доступом:

- добавление по числовому Telegram ID;
- подтверждение перед добавлением;
- восстановление заблокированного сотрудника;
- защита от дубликатов;
- назначение администратора;
- изменение рабочего имени;
- синхронизация `ADMIN_IDS` и `ALLOWED_IDS` при каждом запуске.

```text

💰 Финансы              📸 Портфолио
```

Кнопка «Сотрудники» показывается только администраторам.

## Требования

- Windows 10/11 или Linux;
- Python 3.12 или новее;
- интернет;
- Telegram-бот от `@BotFather`.

## Первая установка на Windows

1. Распакуйте архив, например в `C:\FabrikaBot`.
2. Запустите `install_windows.bat`.
3. Откройте созданный файл `.env`.
4. Заполните минимум:

```env
BOT_TOKEN=токен_из_BotFather
TIMEZONE=Europe/Samara
WORK_CHAT_ID=
ADMIN_IDS=ваш_telegram_id
ALLOWED_IDS=
DB_PATH=data/factory_bot.sqlite3
PORTFOLIO_DIR=data/portfolio
```

5. Запустите `run_windows.bat`.
6. В Telegram отправьте боту `/start`.
7. Укажите рабочее имя: `/register Алексей`.

## Обновление с версии 2 без потери данных

Подробная инструкция находится в `UPDATE_V3.md`.

Самое важное:

1. Остановите старого бота.
2. Скопируйте в безопасное место старые `.env` и папку `data`.
3. Распакуйте новые файлы поверх старой папки с заменой.
4. Не удаляйте существующие `.env` и `data`.
5. Запустите `update_windows.bat` или `install_windows.bat`.
6. Запустите `run_windows.bat`.

При первом запуске структура SQLite-базы обновится автоматически. Старые пользователи, заказы, задачи, дела, расходы, доходы, напоминания, комментарии и фотографии сохраняются. Старые заказы отображаются как объекты категории «Другое».

## Настройка доступа

Постоянные администраторы указываются через запятую:

```env
ADMIN_IDS=
```

Сотрудников можно заранее указать:

```env
ALLOWED_IDS=111111111,222222222
```

Либо добавить через:

```text
👥 Сотрудники → ➕ Добавить сотрудника
```

Пользователь должен хотя бы один раз открыть личный чат с ботом. Иначе Telegram не позволит боту отправить ему первое сообщение.

## Основные команды


## Где хранятся данные

```text
data/factory_bot.sqlite3     база данных
data/portfolio/order_*/      локальные копии фотографий объектов
```

Токен и настройки хранятся в `.env`. Не публикуйте этот файл и не отправляйте посторонним.

## Работа в группе

В `.env` укажите `WORK_CHAT_ID`. Узнать ID группы можно командой `/chatid`, отправленной администратором в нужной группе.

Если формы заполняются в группе, в `@BotFather` отключите Privacy Mode:

```text
/setprivacy → выбрать бота → Disable
```

Безопаснее создавать объекты и финансовые записи в личном чате с ботом.

## Частые ошибки

### Бот пишет «Доступ запрещён» владельцу

Проверьте числовой ID в `ADMIN_IDS`, сохраните `.env` и полностью перезапустите бота.

### Бот работает лично, но не в группе

Проверьте `WORK_CHAT_ID`, права бота и Privacy Mode. После изменения `.env` перезапустите процесс.

### `Conflict: terminated by other getUpdates request`

Одновременно запущены две копии бота. Оставьте только одно окно или один серверный процесс.

### `ModuleNotFoundError`

Повторно запустите `install_windows.bat`.
