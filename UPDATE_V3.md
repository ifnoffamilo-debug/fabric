# Обновление до версии 3.1.0

1. Остановите бота: `Ctrl+C`.
2. Сохраните копию файла `.env` и папки `data`.
3. Распакуйте архив обновления в существующую папку бота с заменой файлов.
4. Не удаляйте `.env` и `data`.
5. Запустите `update_windows.bat`.
6. После проверки запустите `run_windows.bat`.

Обновление базы выполняется автоматически. Старые данные и одиночные фотографии чеков сохраняются.

Подробности находятся в `UPDATE_V3_1.txt` и `CHANGELOG.md`.
